§ 05

Rules

Editorial discipline, codified. Re-read these before introducing a new component or motif.

Do

  • Use one of the five palette tokens. Never a free hex.
  • Set everything in Georgia. Reach for the monospace eyebrow only for utility labels.
  • Derive new motifs from the emblem (head ring, chevron arm) before inventing new shapes.
  • Use one motion easing — --ease-settle.
  • Keep body measure 60–70 characters per line.
  • Use aria-current="page" on the active nav link.
  • Mark all decorative SVG with aria-hidden="true".
  • Use the logo (small) or whitespace for section transitions.

Don’t

  • Use the chevron / wedge as a section divider.
  • Add a second typeface, even “just for headlines”.
  • Round any corner — every input, plate, and frame is square.
  • Use a drop shadow.
  • Introduce a gradient.
  • Make motion bouncy or use ease-in-out.
  • Use Burgundy as a fill behind dark body text.
  • Add a new colour, even “subtly”. Use opacity on existing colours instead.
